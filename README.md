# Fact1 auto care java project

Fact1 Auto Care is a comprehensive JavaFX-based desktop application designed to streamline vehicle service operations for workshops and service centers. It allows administrators and staff to efficiently manage job scheduling, billing, and customer communication.



https://github.com/user-attachments/assets/6d6dac8d-1c30-4fa0-a5d3-ccc9f61c1bad




Key Features:

- User Authentication: Only admin and staff can log in. Upon login, an email is automatically sent to the account holder with their credentials.
- Job Scheduling: Admin can schedule, update, and track vehicle service jobs, including full service, diagnostics, and maintenance tasks.
- Dynamic Billing: Admin can apply discounts, calculate final totals, and generate PDF invoices for completed jobs.
- Customer Communication: System automatically sends detailed bills via email to customers after job completion.
- Data Management: All job and customer data is securely stored in MySQL (XAMPP), ensuring easy access and modification.
- Built With: Java, JavaFX, Scene Builder, MySQL

Benefits:
- Streamlines workshop operations
- Improves customer experience with automated billing and notifications
- Provides a professional, end-to-end solution for vehicle service management

This project demonstrates practical experience in desktop application development, database integration, and automation of workflows.
